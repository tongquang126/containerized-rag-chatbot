Tool Name: jstatus3.py

Description: Display information about an instance.

Usage:
jstatus3.py [-h] [-i INSTANCE] [-e ENVIRONMENT] [--profile PROFILE] [-co CASE] [-sb] [-v] [-c COLOR] [--logs] [--salesforce] [--atlas] [--servicenow] [--quick] [--noninteractive] [--history]

Options:
-h, --help: show this help message and exit
-i INSTANCE, --instance INSTANCE: Specifies the instance name that will be used to gather information. This will match the Salesforce asset name.
-e ENVIRONMENT, --environment ENVIRONMENT: Specify an atlas environment
--profile PROFILE: Specifies the profile to use from your ~/.aws/credentials file.
-co CASE, --case CASE: Provide Jira case number for faster execution
-sb, --sandbox: Use the Everest Sandbox account
-v, --verbose: Verbose mode
-c COLOR, --color COLOR: Specify what color the output should be
--logs, -l: Call get-logs.py to get JSS logs for instance and upload to dropbox
--salesforce, -s: Salesforce info mode
--atlas, -a: Atlas info mode
--servicenow, -sn: ServiceNow info mode
--quick, -q: Quick mode
--noninteractive: Skip any prompts and use default values
--history: Show customization history
