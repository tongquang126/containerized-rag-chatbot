Tool Name: migration_invite.py

Description: Create a SharePoint invitation for a cloud migration and send an invitation email

Required variables (will prompt for if not found):
- JAMF_SSH_USER [ldap, str]
  example: first.last
- JAMF_PASS [ldap, str]
  example: mySecr3tPassw0rdForLDAP
- jiraToken [jira, str]
  example: 123jfkesneasl3130nmslAF
- oauth_token [dropbox, str]
  example: xEECXTJZzevABBBBBBBem3m7-uPsxSpJn3nXokcmKrgpEEl3mw0wZLNFzejKdFH004B3

Usage:
migration_invite.py [-h] [-i INSTANCE] [-e ENVIRONMENT] [--profile PROFILE] [-co CASE] [-v] [-c COLOR] [-s]

Options:
-h, --help: show this help message and exit
-i INSTANCE, --instance INSTANCE: Specifies the instance name that will be used to gather information. This will match the Salesforce asset name.
-e ENVIRONMENT, --environment ENVIRONMENT: Specify an atlas environment
--profile PROFILE: Specifies the profile to use from your ~/.aws/credentials file.
-co CASE, --case CASE: Provide Jira case number for faster execution
-v, --verbose: Verbose mode
-c COLOR, --color COLOR: Specify what color the output should be
-s, --sharepoint: Use this flag to upload the database to SharePoint instead of Dropbox (Will become default after Dropbox deprecation)
